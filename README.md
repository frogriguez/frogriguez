## About this profile

The stats above are generated by a [GitHub Action](.github/workflows/update-stats.yml) that runs every Monday.
It queries the GitHub API with a scoped PAT, aggregates commit and LOC counts, and commits a [`stats.json`](stats.json) file.
A second step in the same job rewrites this README between the comment markers.
